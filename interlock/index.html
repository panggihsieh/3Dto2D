    使用流程

    SVG 接榫
    1. 匯入 SVG，或下載範例 SVG 後再匯入。
    2. 設定內尺寸、材料厚度、Kerf、接榫寬度與深度。
    3. 需要時用 Top tag / Bottom tag 標註上下片。
    4. 按「選取接榫邊」後，先選凸榫 f，再選配對凹榫 F。
    5. 按「確認產生接榫」後，才會顯示檢測報告與 3D 模型，並可輸出 SVG 或 DXF。
