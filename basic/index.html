使用者簡易操作手冊

Basic
  1. 選擇模型類型，例如長方體、正立方體或雙斜屋頂小屋。
  2. 輸入長、寬、高等尺寸，並確認目前使用的是內尺寸。
  3. 設定材料厚度、Kerf 與零件間距，讓切割尺寸符合實際材料。
  4. 需要接榫時按「顯示接榫後」，預覽會更新紅色切割線。
  5. 檢查右側預覽與提示，確認沒有警告後下載 SVG 或 DXF。
